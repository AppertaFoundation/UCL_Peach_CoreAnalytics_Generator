Use-cases

Generator Use-cases:

No. Actor Description
1 Peach-Architecture Tester The tester has a copy of real medical data. He needs to transform the copy to be anonymous for future testing in the main data stream. He chooses the ‘Anonymise data’ option and uploads the file in the generator. He is able to view the file and choose the data field which should be anonymised. Then he clicks the ‘Generate’ button to generate a new file.
2 Peach-Architecture Tester The tester has a copy of a data file. He needs more similar data as the original one. He chooses the ‘Generate novel data’ option and uploads the file in the generator. He is able to view the file and choose the data field which he wants to have more new data. Then he clicks ‘Generate’ button to generate a new file.
