Security Testing

The DC/OS system is running in the compute cluster provided by Microsoft azure, which includes several virtual machines and databases. As the potential data will be the private medical records, we should ensure that the cloud computing system has a high security level. Therefore, we performed penetrating test on the public IPs of our running system. The results help us to change the configuration of the network security.
